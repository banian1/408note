# 路由
## RIP 协议 & 距离向量算法
RIP 协议好消息传得快，坏消息传得慢
RIP特点：当网络出现故障时，要经过比较长的时间（例如数分钟）才能将此信息传送到所有的路由器，“收敛慢”
